# LandingPageAboutMe 🚀

¡Bienvenido/a a mi primer proyecto del **Bootcamp de Desarrollo Web Full Stack en Factoría F5**! Este proyecto consiste en el diseño y desarrollo de una Landing Page de presentación personal ("Sobre mí") construida desde cero utilizando tecnologías web nativas.

El objetivo principal es demostrar el dominio de la estructura semántica en **HTML5**, la estilización personalizada con **CSS3** (sin el uso de librerías o frameworks externos), el control de versiones con **Git** y el despliegue a producción a través de **GitHub Pages**.

---

## 🎯 Requisitos del Proyecto

El proyecto se ha desarrollado siguiendo estrictamente las directrices y pautas solicitadas:

### 🏛️ Estructura Semántica y Arquitectura
La página cuenta con una estructura limpia y bien organizada dividida en tres bloques principales:
- **`Header`**: Barra de navegación y título principal.
- **`Main`**: Contenedor principal de la información y secciones de presentación.
- **`Footer`**: Información de copyright, enlaces de contacto y redes sociales.

### 📑 Secciones Incluidas
Cada sección combina texto descriptivo con imágenes optimizadas generadas mediante Inteligencia Artificial (**IAgen**):
1. **Presentación Principal**: Nombre, apellidos y foto de perfil.
2. **Origen**: Lugar de nacimiento y nacionalidad.
3. **Pasatiempos (Hobbies)**: Qué me apasiona hacer en mi tiempo libre.
4. **Formación y Estudios**: Qué he estudiado y los centros donde me he formado.
5. **Trayectoria**: Un breve recorrido por mi historia, experiencia y motivaciones.

### 🎨 Estilos y Diseño (CSS Puro)
- **Sin Librerías**: No se permite el uso de frameworks ni librerías CSS externas (como Bootstrap, Tailwind, etc.). Todo el diseño es CSS nativo.
- **Estructura y Maquetación**: Cuidado minucioso en la estructura del Layout, espaciados (`margin`, `padding`), alineaciones y uso de tipografías.
- **Diseño Responsivo**: Adaptabilidad básica para asegurar que la página sea legible en diferentes pantallas.

---

## 🛠️ Tecnologías Utilizadas

- **HTML5**: Estructuración semántica del contenido.
- **CSS3**: Estilos personalizados, colores, layouts y tipografías.
- **IAgen**: Herramientas de Inteligencia Artificial para la generación de imágenes personalizadas.
- **Git & GitHub**: Control de versiones y alojamiento del código fuente.
